<p>This class represents an agent in the platform BESA. An agent consists of a concurrent behaviors group, that has associate guards as well.  In order to construct new agents it is necessary to create a class that extends of AgentBESA. <br /></p><p>@since JDK1.0<br /></p>

public abstract
setupAgent():void
Details:
Sequential
Notas: Starting routine of the agent. Invoked automatically before the infinite cycle. To consider when implementing this method:  - The state was initialized previously in the constructor of the agent. - Create the guards (automatic bind). - Create the behaviors (automatic registry). - Register agent services.  It must be implemented by the user to create an agent.
